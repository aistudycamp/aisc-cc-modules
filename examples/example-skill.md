# Daily Standup

Help the user run a quick daily standup check-in.

## Steps

1. Ask: "What did you work on yesterday (or last session)?"
2. Ask: "What are you planning to work on today?"
3. Ask: "Is anything blocking your progress?"
4. Format their answers into a clean standup summary with today's date.

## Tone
Keep it casual and encouraging. This is a quick check-in, not a formal report.
