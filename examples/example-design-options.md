# Design Challenge — Option Reference

Choose one of these creative options for your design assignment.

## Option A: HTML Slide Deck
Build a polished presentation that opens in your browser.
- **You'll get**: An HTML file with slides, transitions, and styling
- **Good for**: Presenting ideas, teaching others, summarizing projects
- **No special tools needed** — Claude builds it with standard HTML/CSS

## Option B: Excalidraw Diagram
Create a visual diagram on a live canvas — flowcharts, process maps, infographics.
- **You'll get**: A visual diagram you can export or screenshot
- **Good for**: Visual thinkers, explaining processes, mapping systems
- **Uses**: `excalidraw-skill`

## Option C: Polished Web Page
Build a beautiful, modern web page — landing page, portfolio, or dashboard.
- **You'll get**: A complete HTML/CSS page ready to view in your browser
- **Good for**: Seeing what professional web design looks like with Claude
- **Uses**: `/frontend-design` skill

## Option D: AI Image Prompt
Craft a detailed, expert-level prompt for Midjourney, DALL-E, or another image tool.
- **You'll get**: A polished prompt you paste into your image generation tool of choice
- **Good for**: Marketing visuals, creative exploration, learning prompt engineering
- **No special tools needed** — you run the prompt yourself

## Option E: Interactive Playground
Build an interactive HTML widget — calculator, quiz, explorer, decision tool.
- **You'll get**: A self-contained interactive HTML file
- **Good for**: People who want something functional and clickable
- **Uses**: `/playground` skill
